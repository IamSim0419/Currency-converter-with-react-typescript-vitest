# React + TypeScript + Vite + Vitest for testing + Tailwind CSS Currency Converter

A simple currency converter application built with React, TypeScript, and Vite. It allows users to convert currencies, view conversion history, and copy results to the clipboard.

## Features

- Convert currencies using real-time exchange rates from an external API.
- Save conversion history and view it in a modal.
- Copy conversion results to the clipboard.
- Responsive design using Tailwind CSS.
- Tested with Vitest for reliable functionality.
